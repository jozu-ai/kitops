# kit Commands

These functions provide programmatic access to the KitOps Command-Line Interface.

Available commands include:

* `info()`
* `inspect()`
* `list()`
* `login()`
* `logout()`
* `pack()`
* `pull()`
* `push()`
* `remove()`
* `tag()`
* `unpack()`
* `version()`

You can import them all from `kitops.cli.kit`

```python
import kitops.cli.kit as kit
```
